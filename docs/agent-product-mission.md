# Agent Product Mission — Псё

## Product North Star

Псё — ассистент по владению и воспитанию собаки. Аватар — catchy onboarding hook, не ядро продукта.

## Core user promise

> “Я лучше понимаю, как заботиться о своей собаке сегодня: что не забыть, как воспитывать, где гулять, с кем знакомиться, когда идти к врачу, что купить.”

## Current state

- Next.js app-like MVP deployed on Vercel.
- Supabase project connected.
- Schema + RLS applied.
- Demo-owner persistence exists via `/api/pets` and `/api/app/bootstrap`.
- UI tabs: Today / Passport / Assistant / Map / Wishlist.
- Not yet real auth.
- Not yet real assistant LLM flow.
- Not yet real reminders CRUD in UI.
- Not yet real map or partner integrations.

## Product pillars

1. Паспортичка
2. Карта / зоны
3. Социализация
4. Ассистент
5. Напоминания
6. Клиники / ветшопы / подборки
7. Вишлисты
