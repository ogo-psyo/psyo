# Псё — Vercel MVP

Mobile-first web/PWA prototype for daily dog care, walks, map markers, presence, and nearby-compatible dogs.

## Local

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## MVP scope

- One pet profile
- Daily state logs saved in localStorage
- Care calendar mock
- Dog map marker mock
- Live presence toggle
- Nearby dogs / compatibility cards

## Later

- Supabase auth/database
- Real maps
- Push notifications
- Moderation/trust for UGC markers
